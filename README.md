# Pizza Sales Data Analysis using SQL

## Project Overview
This project analyzes a pizza sales dataset using SQL to extract meaningful business insights.  
The goal is to understand customer ordering behavior, identify top-selling pizzas, and analyze revenue trends.

This project demonstrates practical SQL skills such as joins, aggregation, grouping, ranking functions, and window functions.

## Objectives
The analysis answers several business questions such as:

- Total number of orders placed
- Total revenue generated from pizza sales
- Highest priced pizza
- Most common pizza size ordered
- Top 5 most ordered pizza types
- Category-wise pizza distribution
- Orders distribution by hour of the day
- Average number of pizzas ordered per day
- Top 3 pizzas based on revenue
- Percentage contribution of each pizza type
- Cumulative revenue over time
- Top 3 pizzas by revenue for each category

## Dataset Tables
The project uses multiple tables:

- orders
- order_details
- pizzas
- pizza_types

These tables are joined together to generate insights.

## SQL Concepts Used
This project demonstrates the use of:

- SELECT statements
- JOINS
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- Subqueries
- Window Functions
- Ranking Functions

## Key Insights
Some important insights obtained from the analysis:

- Identification of the most popular pizza types
- Revenue contribution of different pizza categories
- Customer ordering trends by hour of the day
- Cumulative revenue growth over time
- Best performing pizzas by revenue
